---
title: Button groups
description: Combine related .btn elements into a single segmented row or column. Same DOM from either package.
---

import { Button, ButtonGroup } from "@aortl/admin-react";

A button group renders multiple `.btn` children as one visually combined unit: shared borders, rounded outer corners, square inner corners. It is presentational only — each child is still an independently focusable button. For single-select toggles, reach for a dedicated toggle / segmented control instead.

## Examples

### Horizontal (default)

:::example

```html
<div class="btn-group">
  <button class="btn btn-secondary">Day</button>
  <button class="btn btn-secondary">Week</button>
  <button class="btn btn-secondary">Month</button>
</div>
```

```tsx
<ButtonGroup>
  <Button variant="secondary">Day</Button>
  <Button variant="secondary">Week</Button>
  <Button variant="secondary">Month</Button>
</ButtonGroup>
```

:::

### Vertical

:::example

```html
<div class="btn-group btn-group-vertical">
  <button class="btn btn-secondary">Up</button>
  <button class="btn btn-secondary">Center</button>
  <button class="btn btn-secondary">Down</button>
</div>
```

```tsx
<ButtonGroup orientation="vertical">
  <Button variant="secondary">Up</Button>
  <Button variant="secondary">Center</Button>
  <Button variant="secondary">Down</Button>
</ButtonGroup>
```

:::

### Split action

A primary action plus a related secondary action read as one unit.

:::example

```html
<div class="btn-group">
  <button class="btn btn-primary">Save</button>
  <button class="btn btn-secondary">Save and add another</button>
</div>
```

```tsx
<ButtonGroup>
  <Button>Save</Button>
  <Button variant="secondary">Save and add another</Button>
</ButtonGroup>
```

:::

### Sizing

Set the size on each child, not the group. Keep all children the same size for a clean group.

:::example

```html
<div class="btn-group">
  <button class="btn btn-secondary btn-sm">Sort</button>
  <button class="btn btn-secondary btn-sm">Filter</button>
  <button class="btn btn-secondary btn-sm">Export</button>
</div>
```

```tsx
<ButtonGroup>
  <Button variant="secondary" size="sm">
    Sort
  </Button>
  <Button variant="secondary" size="sm">
    Filter
  </Button>
  <Button variant="secondary" size="sm">
    Export
  </Button>
</ButtonGroup>
```

:::
