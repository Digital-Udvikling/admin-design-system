`RadioGroup` provides the selection state for its child `Radio`s. The React side wraps Base UI's `Radio.Root` (rendered as a `<span>` with a hidden `<input>`); vanilla styles a native `<input type="radio">` with the same class names.

## Examples

### Group (horizontal)

<Example
  html={`<div class="radio-group" role="radiogroup">
  <label style="display: inline-flex; align-items: center; gap: 0.5rem;">
    <input type="radio" name="size" value="sm" class="radio"> Small
  </label>
  <label style="display: inline-flex; align-items: center; gap: 0.5rem;">
    <input type="radio" name="size" value="md" class="radio" checked> Medium
  </label>
  <label style="display: inline-flex; align-items: center; gap: 0.5rem;">
    <input type="radio" name="size" value="lg" class="radio"> Large
  </label>
</div>`}
  react={`<RadioGroup name="size" defaultValue="md">
  <label style={{ display: "inline-flex", alignItems: "center", gap: "0.5rem" }}>
    <Radio value="sm" /> Small
  </label>
  <label style={{ display: "inline-flex", alignItems: "center", gap: "0.5rem" }}>
    <Radio value="md" /> Medium
  </label>
  <label style={{ display: "inline-flex", alignItems: "center", gap: "0.5rem" }}>
    <Radio value="lg" /> Large
  </label>
</RadioGroup>`}
>
  <RadioHorizontalDemo client:load />
</Example>
