The form-control set: text entry, choice, toggle, composition.

- **[Inputs](./inputs/)** — single-line text, email, password, number, date, file, …
- **[Textareas](./textareas/)** — multi-line text entry.
- **[Selects](./selects/)** — single-choice dropdown with a custom popup.
- **[Checkboxes](./checkboxes/)** — independent on/off toggles for booleans and lists.
- **[Radios](./radios/)** — mutually exclusive choice within a `RadioGroup`.
- **[Switches](./switches/)** — immediate on/off settings; for form-submission booleans prefer a checkbox.
- **[Fields](./fields/)** — labels, descriptions, and validation messaging wrapping any of the above.

Each control emits the same class names from `@aortl/admin-css` (vanilla) and `@aortl/admin-react` (React wrappers around Base UI). Drop them into a `Field` for accessible label/error wiring out of the box.
