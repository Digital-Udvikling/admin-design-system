---
title: Buttons
description: Base class .btn + a variant + an optional size. Identical DOM from either package.
---

import { Button } from "@aortl/admin-react";
import { IconPlus, IconTrash, IconDownload, IconDotsVertical } from "@tabler/icons-react";

## Examples

### Variants

:::example

```html
<button class="btn btn-primary">Primary</button>
<button class="btn btn-secondary">Secondary</button>
<button class="btn btn-ghost">Ghost</button>
<button class="btn btn-danger">Danger</button>
```

```tsx
<Button variant="primary">Primary</Button>
<Button variant="secondary">Secondary</Button>
<Button variant="ghost">Ghost</Button>
<Button variant="danger">Danger</Button>
```

:::

### Sizes

:::example

```html
<button class="btn btn-primary btn-sm">Small</button>
<button class="btn btn-primary">Medium</button>
<button class="btn btn-primary btn-lg">Large</button>
```


### With icons

Pass an icon component to `icon` for a leading icon, or `iconTrailing` for a trailing one. The wrapper renders it at `size={16}` with `aria-hidden`. For an icon-only button, pass just `icon` and add an `aria-label`. See [Icons](/admin-design-system/theme/icons/) for the recommended library.

:::example

```html
<button class="btn btn-primary">
  <i class="ti ti-plus" aria-hidden="true"></i>
  New order
</button>
<button class="btn btn-secondary">
  Export
  <i class="ti ti-download" aria-hidden="true"></i>
</button>
<button class="btn btn-ghost" aria-label="More actions">
  <i class="ti ti-dots-vertical" aria-hidden="true"></i>
</button>
<button class="btn btn-danger">
  <i class="ti ti-trash" aria-hidden="true"></i>
  Delete
</button>
```

```tsx
<Button variant="primary" icon={IconPlus}>New order</Button>
<Button variant="secondary" iconTrailing={IconDownload}>Export</Button>
<Button variant="ghost" icon={IconDotsVertical} aria-label="More actions" />
<Button variant="danger" icon={IconTrash}>Delete</Button>
```

:::
