Override design tokens via CSS variables. Works the same whether you use the vanilla CSS or the React package.

## Override tokens

In your own stylesheet, loaded _after_ the design system:

```css
:root {
  --color-primary: oklch(0.6 0.2 30);
  --radius-control: 0.25rem;
}
```
