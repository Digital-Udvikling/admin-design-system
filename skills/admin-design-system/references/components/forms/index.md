# Forms

> Input controls and composition primitives.

- **[Inputs](inputs.md)** — single-line text, email, password, number, date, …
- **[Textareas](textareas.md)** — multi-line text.
- **[Selects](selects.md)** — single-choice dropdown with a custom popup.
- **[Checkboxes](checkboxes.md)** — independent on/off toggles.
- **[Radios](radios.md)** — mutually exclusive choice within a `RadioGroup`.
- **[Switches](switches.md)** — immediate on/off settings (for form submissions, prefer a checkbox).
- **[File inputs](file-inputs.md)** — file picker styled to match other controls.
- **[Input groups](input-groups.md)** — prefix, suffix, or action button attached to an input.
- **[Fields](fields.md)** — labels, descriptions, and validation messaging.
