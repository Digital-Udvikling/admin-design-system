The React `Select` is a Base UI compound primitive — `Select.Trigger`, `Select.Popup`, `Select.Item` — that renders a custom popup. The vanilla example uses a native `<select>` styled with `.select-trigger` so the class-name contract still holds, even though the popup chrome can only be customized in React.

## Examples

### Default

<Example
  html={`<select class="select-trigger select-trigger-bordered">
  <option value="">Select a status…</option>
  <option value="open">Open</option>
  <option value="in-progress">In progress</option>
  <option value="closed">Closed</option>
</select>`}
  react={`<Select name="status">
  <Select.Trigger>
    <Select.Value placeholder="Select a status…" />
    <Select.Icon />
  </Select.Trigger>
  <Select.Popup>
    <Select.Item value="open">
      Open
      <Select.ItemIndicator />
    </Select.Item>
    <Select.Item value="in-progress">
      In progress
      <Select.ItemIndicator />
    </Select.Item>
    <Select.Item value="closed">
      Closed
      <Select.ItemIndicator />
    </Select.Item>
  </Select.Popup>
</Select>`}
>
  <SelectDefaultDemo client:load />
</Example>

### Inside a Field

<Example
  html={`<div class="field">
  <label class="field-label" for="role">Role</label>

    <option value="admin">Admin</option>

  </select>

    </Select.Popup>
  </Select>

</Field>`}
