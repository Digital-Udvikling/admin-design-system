### Types

`Input` forwards the native `type` attribute, so every HTML input type works — the browser handles affordances like date pickers, number steppers, and password masking.

<Example
  html={`<input class="input input-bordered" type="email" placeholder="you@example.com">
<input class="input input-bordered" type="password" placeholder="Password">
<input class="input input-bordered" type="number" placeholder="42">
<input class="input input-bordered" type="date">
<input class="input input-bordered" type="time">
<input class="input input-bordered" type="search" placeholder="Search">
<input class="input input-bordered" type="url" placeholder="https://example.com">
<input class="input input-bordered" type="tel" placeholder="+45 12 34 56 78">`}
  react={`<Input type="email" placeholder="you@example.com" />
<Input type="password" placeholder="Password" />
<Input type="number" placeholder="42" />
<Input type="date" />
<Input type="time" />
<Input type="search" placeholder="Search" />
<Input type="url" placeholder="https://example.com" />
<Input type="tel" placeholder="+45 12 34 56 78" />`}
>
  <div
    style={{
      display: "flex",
      flexDirection: "column",
      gap: "0.75rem",
      width: "100%",
      maxWidth: "24rem",
    }}
  >
    <Input type="email" placeholder="you@example.com" />
    <Input type="password" placeholder="Password" />
    <Input type="number" placeholder="42" />
    <Input type="date" />
    <Input type="time" />
    <Input type="search" placeholder="Search" />
    <Input type="url" placeholder="https://example.com" />
    <Input type="tel" placeholder="+45 12 34 56 78" />
  </div>
</Example>

### File

`type="file"` keeps the browser's native chooser button — pair it with a `Field.Label` (see [Fields](../fields/)) for accessible labeling.

<Example
  html={`<input class="input input-bordered" type="file">
<input class="input input-bordered" type="file" accept="image/*" multiple>`}
  react={`<Input type="file" />
<Input type="file" accept="image/*" multiple />`}
>
  <div
    style={{
      display: "flex",
      flexDirection: "column",
      gap: "0.75rem",

    <Input type="file" accept="image/*" multiple />
  </div>
</Example>
